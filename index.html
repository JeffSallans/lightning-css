Positioning

Block vs Inline

Inline
take only as much space as they need,
no new line after use





Block
spanning the whole width of the available space,
new line after use

HTML Element Default

Inline - span, a, strong, em, img, br, input, abbr, acronym

Block - div, h1...h6, p, ul, li, table, blockquote, pre, form

Absolute vs Relative






Absolute
element will be positioned in reference to parent element with position property defined or document body
Relative
element will be positioned after the last relative sibling element

Work inside a container
